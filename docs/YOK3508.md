# 三支路合唱数字单块设计（代号：YOK3508）

在音频效果器里，合唱 (Chorus) 是一种经典的空间扩展类效果。它通过叠加带有轻微延迟和音调变化的信号副本，模拟多个声源同时发声的听感。本文将对一个基于JUCE框架实现的三支路立体声合唱算法进行详细拆解，阐述其背后的数字信号处理 (DSP) 逻辑、关键参数对声音的具体影响，以及多支路架构带来的声场优势。

## 声音的立体感

### 第一：时间轴偏移

单支路的合唱效果器通过一个LFO（低频振荡器）调制延迟时间，产生单一的音调波动感。这种处理虽然能增加厚度，但声像较为居中，缺乏宽广的立体声像。

本设计采用了三个独立的调制支路（Chorus Branch），分别处理输入信号并输出到左右声道。这种架构并非简单的三倍叠加，而是通过相位偏移 (Phase Offset) 和通道间失谐 (Detune) 来构建一个复杂的、具有纵深感的立体声场。单支路和三支路音响效果对比如下：

在直观感受上，立体声的宽度源于左右声道信号的差异。差异越大，声场越宽。然而，如果这种差异处理不当——例如简单地将左声道复制并延迟几毫秒放到右声道——听者会立刻察觉到声像严重偏向某一侧，产生令人不适的“侧倾感”。三支路合唱设计通过精心编排的 毫秒级时间偏移 规避了这一陷阱，其背后的声学依据便是哈斯效应 (Haas Effect)。

哈斯效应是心理声学中关于优先声级定位的经典理论，其核心结论可概括为：

**时间差决定方向**：当两个强度相同的相似声音先后到达双耳，且时间差在 1ms 至 30ms 之间时，听者会明确感知到声源来自最先到达的那个方向。人脑会将后到达的声音抑制为一种“空间印象”，而不会分辨出两个独立声源。人的双耳听到极其明确的“极左”或“极右”方位（ITD），通常需要 0.7 到 1.5 毫秒的时间差（因为声音在空气中传播从左耳到右耳大约需要0.6~1.0ms）。而如果要产生强烈的“哈斯宽广包围感”，需要 15 到 30 毫秒的时间差(15 毫秒相当于声音在空气中走过了大约 5 米的距离,心理声学认为这五米的距离会让大脑觉得声音是远处反射的回声)。

[这里放置只有左声场的音频-1.0ms] vs [这里放置只有左声场的音频-15ms]

合唱效果器中的支路时间偏移量恰好落入了哈斯效应的工作区间。在本设计中，三支路的调度逻辑如下：

```cpp
    //处理第一组声道（0和1），相位偏移为0
    processCertainChorus(buffer, 
        0.0f,//时间轴毫秒数偏移
        0.0f, //LFO信号轴相位偏移
        chorusBranch1, 
        startSample, 
        numSamples);

    //相位偏移为 +currentPhaseOffsets
    processCertainChorus(buffer, 
        currentPhaseOffsetMs, ... );

    //相位偏移为-currentPhaseOffsetMs
    processCertainChorus(buffer, 
        -currentPhaseOffsetMs, ... );
```

以典型调制速率和深度计算，这些相位偏移折算为时间差通常在 2ms 至 8ms 范围内。根据哈斯效应，听者的听觉中枢会将时间领先的支路判定为声像定位的参考，而将延迟到达的支路融入混响感的背景中。

### 第二：信号轴偏移

把左声道和右声道的 LFO 想象成两个正在揉弦的吉他手。调整它们之间的相位差，会产生以下几种截然不同的物理现象和听觉作用：

1. **相位差0°（绝对同向）**；左耳和右耳的延迟时间同时变长、同时变短。这意味着左右声道的音高在同时升高、同时降低,让人感觉声场锁死在正中间，没有任何立体声广度
2. **相位差180°（绝对反向）**：左耳延迟变长时，右耳延迟正在变短。这意味着当左耳听到音高变低（降号）时，右耳听到的是音高变高（升号），形成一种撕裂感。但是如果在单声道设备上播放音频的时候，很容易造成相干抵消。
3. **相位差90°**：当左边音高变化最剧烈的时候，右边刚好处于波峰或波谷（音高暂时不变化），这样的话，声音不再是简单的左右推拉，而是产生了一种的流动感。就像海浪一样，声音从左边滚到右边，柔和、平滑且自然。

而在代码中，我们实现信号轴偏移的方法如下：

```c++
float sineValueLeft = std::sin(sineIndexLeft);
float sineIndexRight = sineIndexLeft + rightRadToLeftRad ;
float sineValueRight = std::sin(sineIndexRight);

//而后再进入双耳延迟时间中
float leftDelayMs = currentBaseDelayMs + currentDepthMs * sineValueLeft + phaseOffsetMs / 2;
float rightDelayMs = currentBaseDelayMs + currentDepthMs * sineValueRight - phaseOffsetMs / 2;
```

## 双耳延迟时间的不同

合唱效果器的核心在于双耳延迟时间的差异：

```c++
float leftDelayMs = currentBaseDelayMs + currentDepthMs * sineValueLeft + phaseOffsetMs / 2;
float rightDelayMs = currentBaseDelayMs + currentDepthMs * sineValueRight - phaseOffsetMs / 2;
```

式中，left/rightDelayMs指的是两个通道延迟的毫秒数，currentBaseDelayMs代表基准延迟时间，currentDepthMs指LFO调制的振幅，sineValueLeft/sineValueRight指的是LFO的相位，phaseOffsetMs代表声音的方位，下面具体讲解每个参数的作用

### currentBaseDelayMs(基准延迟时间)

延迟时间的时域公式为： $ y[n] = x[n] + 调制系数 * x[n - delayMs] $

计算它的傅里叶变换得到： $ H(z) = 1 + 调制系数 * z^{-delayMs} $

得到频谱振幅：$ |H(z)| = \sqrt{1 + {调制系数}^2 + 2 * 调制系数 * cos(\omega * delayMs)} $

可以清晰的看出，频谱振幅是一个梳状滤波器，而他的振动速度和延迟时间紧密相关，而在延迟时间中，基础延迟时间占据了绝大部分。

在听感上：

- 短基准延迟（1~5 ms）：梳齿稀疏，在可听频段内只有几个明显的峰谷。随着 LFO 调制，这些峰谷扫过频谱，产生类似“镶边”的金属感、扫频感。此时效果更接近 Flanger，而非典型的 Chorus。

- 长基准延迟（25~50 ms）：梳齿极密（间距 20~40 Hz），峰谷几乎连成一片平滑的频谱包络。人耳听不到单独的梳状滤波声，只感觉到声音变厚、变丰满，并且由于延迟超过 Haas 效应的阈值（约 30 ms），会产生清晰的“双耳回声”感知，但又不至于独立成两个音。这是最典型的合唱效果区间。

### rate 和 currentDepthMs

根据信号与系统知识，时域的群时延速度会导致频域的相位移动，当正弦波震荡的速度(rate)越快，音高变化的速度越快，当正弦波的振幅越大（currentDepthMs），音高变化效果越强。

## 反馈度

当反馈量被推高至 50% 以上时，合唱效果器会突然表现出一种极具辨识度的科幻音色——类似外星飞船引擎的金属嗡鸣，或是老式电子游戏中的激光枪扫射声。

[放置无反馈和高反馈的对比]

从信号与系统的角度看，单支路合唱处理单元构成了一个典型的无限脉冲响应 (IIR) 梳状滤波器。时域的变化函数为：
$$ 
y[n] = x[n] + feedback * y[n - delayMs] 
$$
而其系统函数可以表示为

$$
H(z) = \frac{z^{-delayMs}}{1 - feedback * z^{-delayMs}}
$$
